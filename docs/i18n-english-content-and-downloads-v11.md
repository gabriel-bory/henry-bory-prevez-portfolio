# v11 — English content foundation and downloadable CV assets

## Objetivo

Avanzar la rama `web/011-i18n-english-content` con dos mejoras:

1. Contenido inicial real en inglés para páginas principales.
2. Descargas pendientes incorporadas al sitio.

## Descargas añadidas

### Español

- CV de una página: PDF + DOCX.
- Dossier profesional: PDF + DOCX.

### English

- Executive CV: PDF + DOCX.
- Academic CV 2025: PDF fuente.
- ATS CV: PDF + DOCX.

## Nota editorial

Los documentos generados son una base profesional inicial. El CV académico inglés fuente conserva el PDF aportado como evidencia. Las versiones ejecutiva/ATS en inglés y el resumen/dossier español deben seguir refinándose con evidencia documental cuando corresponda.
