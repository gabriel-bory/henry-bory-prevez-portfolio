# v14 — Estrategia internacional de CVs e idiomas

## Objetivo

Ampliar el portafolio de Henry Bory Prevez a siete idiomas:

- Español (`es`)
- English (`en`)
- Nederlands (`nl`)
- Português (`pt`)
- Français (`fr`)
- Deutsch (`de`)
- Italiano (`it`)

La prioridad editorial se mantiene así:

1. Español: fuente principal y base de evidencia.
2. Inglés: idioma internacional prioritario.
3. Portugués, francés, alemán e italiano: estructura preparada con disponibilidad controlada.
4. Neerlandés: estructura activa, con traducción pendiente de revisión.

## Familias de documentos

La web distingue cinco familias de documentos:

1. CV académico completo.
2. CV ejecutivo internacional.
3. CV ATS.
4. CV de una página.
5. Dossier profesional.

## Política de foto

- CV ejecutivo, CV de una página y dossier: pueden usar la foto oficial de CV.
- CV académico completo: foto opcional si se crea portada o encabezado institucional.
- CV ATS: sin foto, sin columnas complejas, sin elementos decorativos.

## Estado actual

- ES: documentos disponibles.
- EN: ejecutivo y ATS disponibles; CV académico fuente disponible en PDF; one-page y dossier en revisión.
- NL/PT/FR/DE/IT: estructura y disponibilidad publicadas como “en preparación”.

## Criterio profesional

No se publican CVs completos traducidos automáticamente como versión final. Cada idioma requiere:

- traducción humana o revisión humana;
- validación terminológica;
- revisión de nombres de títulos académicos;
- conservación de evidencia y trazabilidad;
- adaptación mínima al público objetivo sin alterar méritos.
