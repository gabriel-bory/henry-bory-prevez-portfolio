# Responsive, contenido y navegación — v05

## Objetivo

Corregir el header en anchuras extremas (250–360 px), evitar scroll horizontal en secciones como Software y mejorar el texto conductor de cada página.

## Cambios principales

- `LanguageSelector` visualmente compacto (`ES`, `EN`, `NL`).
- Header más robusto en 250 px: nombre se simplifica, botón menú pasa a icono y controles reducen anchura.
- `overflow-x: clip` y `overflow-wrap: anywhere` para evitar scroll horizontal.
- `PageIntro` y `section-summaries.ts` unifican el resumen de cada sección.
- Página principal con descripción más coherente del perfil académico.

## Validación recomendada

- 250 px
- 280 px
- 320 px
- 360 px
- 390 px
- 430 px
- 768 px
- 1024 px
- 1280 px
