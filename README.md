# Henry Bory Prevez Portfolio

Sitio web profesional del Dr. en Ciencias Henry Bory Prevez, construido con Astro.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Estructura

- `src/pages/`: páginas públicas.
- `src/layouts/`: layouts base.
- `src/components/`: componentes reutilizables.
- `src/content/`: publicaciones, proyectos y software.
- `public/files/cv/`: CVs descargables públicos.
- `public/images/profile/`: imagen de perfil final limpia.

## Nota

Este repositorio debe contener solo contenido público aprobado.
No subir certificados, evidencias privadas ni datos sensibles.
