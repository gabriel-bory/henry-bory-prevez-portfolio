# Henry Bory Prevez Professional Portfolio

Professional, multilingual academic portfolio for **Dr. Henry Bory Prevez**, built with **Astro** and deployed on **GitHub Pages**.

This project presents a public, curated and verifiable professional profile, including academic information, portfolio content, downloadable CV assets and QR-based access to the live website.

## Overview

The goal of this project is to provide a clean, responsive and documentation-oriented portfolio website for an academic and professional profile.

The site is designed to support:

- Public academic profile presentation
- Multilingual content structure
- Downloadable CV and portfolio assets
- Professional software, publications and project sections
- GitHub Pages deployment
- QR code access to the live website
- Long-term maintainability through a structured Astro codebase

---

## Main features

- Responsive static website built with Astro
- Multilingual content structure

- Downloadable CV files and portfolio assets

## Project structure

```text
.
├── .github/workflows/        # GitHub Actions deployment workflow
├── docs/                     # Project documentation
├── public/
│   ├── files/cv/             # Public downloadable CV files
│   ├── images/profile/       # Public profile images
│   └── images/qr/            # QR code assets
├── src/
│   ├── components/           # Reusable UI components
│   ├── content/              # Publications, projects and software content
│   ├── data/                 # Site metadata and shared data
│   ├── layouts/              # Base layouts
│   ├── pages/                # Public routes and pages
│   └── styles/               # Global and component styles
├── astro.config.mjs          # Astro configuration
├── package.json              # Project scripts and dependencies
└── README.md                 # Project documentation
```

## Local development

### Prerequisites

- Node.js installed
- npm installed
- Git installed

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build production version

```bash
npm run build
```

### Preview production build locally

```bash
npm run preview
```

## Public content policy

This repository must contain only public, approved and non-sensitive content.

Do not upload:

- Private certificates
- Personal identification documents
- Non-public academic evidence
- Sensitive personal data
- Unapproved institutional documents
- Private research or administrative files

All downloadable files placed under `public/files/` must be reviewed before publication.
